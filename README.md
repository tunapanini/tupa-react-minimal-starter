# tupa-react-minimal-starter

A minimal React + Typescript start with parcel

[React](https://reactjs.org/) + [Parcel](https://parceljs.org/)

## Start Dev Server (HMR)

```
yarn start
```

## Build

```
yarn build
```

## Lint code \w ESLint

```
yarn lint
```

## Format code \w prettier

```
yarn prettier
```
